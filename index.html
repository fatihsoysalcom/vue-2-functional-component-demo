Vue.js Functional Components Demo (Vue 2)

This example illustrates the difference between a regular, stateful Vue component and a lightweight, stateless functional component.

Regular Vue Component (Stateful)

This component has its own internal data and methods, managing its state independently.

Functional Vue Component (Stateless)

This component is lightweight, has no internal state, and only renders based on the props it receives from its parent.

Watch the functional component update when the parent count changes, as it directly reflects its props.
