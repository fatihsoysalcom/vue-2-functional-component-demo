# Vue 2 Functional Component Demo

This example demonstrates the difference between a regular (stateful) Vue.js component and a functional (stateless) Vue.js component using Vue 2 syntax. It highlights how functional components are lightweight, instance-less, and render solely based on their props, making them suitable for purely presentational tasks. A regular component with internal state and methods is included for comparison.

## Language

`html`

## How to Run

Save the code as `index.html`.
Open `index.html` in your web browser.
